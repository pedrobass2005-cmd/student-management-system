Student Management System

Student Management System é um sistema simples de gerenciamento de alunos desenvolvido em Python, com foco em prática de lógica de programação, CRUD e persistência de dados.

Este projeto simula um sistema real de cadastro de alunos, utilizando arquivos JSON como armazenamento local.

📌 Objetivo

Este projeto foi desenvolvido com foco em aprendizado prático de:

Lógica de programação
Estruturação de sistemas CRUD
Manipulação de arquivos JSON
Organização de código em Python
⚙️ Funcionalidades
📥 Cadastro de alunos
📋 Listagem de alunos
✏️ Atualização de dados
🗑️ Remoção de registros
💾 Persistência de dados em arquivo JSON
🔄 Carregamento automático dos dados ao iniciar o sistema
🧠 Tecnologias utilizadas
Python 3
JSON (armazenamento local)
🚀 Como executar o projeto
# 1. Clone o repositório
git clone https://github.com/pedrobass2005-cmd/student-management-system.git

# 2. Acesse a pasta
cd student-management-system

# 3. Execute o sistema
python main.py
